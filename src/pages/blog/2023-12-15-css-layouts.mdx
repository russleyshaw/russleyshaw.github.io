# Flex

## Flex: Justify Content

```css
justify-content: flex-start; /* default */
justify-content: center;
justify-content: flex-end;
justify-content: space-between;
justify-content: space-around;
```

<div className={styles.card} style={{ display: "flex", flexDirection: "row" }}>
    <div className={styles.flexItem}>flex-start</div>
    <div className={styles.flexItem}>Flex 1</div>
    <div className={styles.flexItem}>Flex 2</div>
    <div className={styles.flexItem}>Flex 3</div>
</div>

<div
    className={styles.card}
    style={{ display: "flex", flexDirection: "row", justifyContent: "center" }}
>
    <div className={styles.flexItem}>center</div>
    <div className={styles.flexItem}>Flex 1</div>
    <div className={styles.flexItem}>Flex 2</div>
    <div className={styles.flexItem}>Flex 3</div>
</div>

<div
    className={styles.card}
    style={{ display: "flex", flexDirection: "row", justifyContent: "space-between" }}
>
    <div className={styles.flexItem}>space-between</div>
    <div className={styles.flexItem}>Flex 1</div>
    <div className={styles.flexItem}>Flex 2</div>
    <div className={styles.flexItem}>Flex 3</div>
</div>

### Flex: Align Items

```css
display: flex;
flex-direction: column;

align-items: stretch; /* default */
align-items: start;
align-items: end;
align-items: center;
align-items: baseline;
```
